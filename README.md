# 🔐 Cyber Security Dashboard

![Status](https://img.shields.io/badge/Status-Active-brightgreen)
![Made With](https://img.shields.io/badge/Made%20With-Flask%20%7C%20JavaScript-blue)
![License](https://img.shields.io/badge/License-Educational-orange)

A modern cybersecurity dashboard that detects suspicious login activity, tracks intrusion attempts, and visualizes attack patterns in real time.

---

## 🚀 Features

✨ Real-time intrusion detection  
🌍 IP address & location tracking  
🚨 Suspicious activity alerts  
🔒 Automatic IP blocking after repeated failures  
📊 Live dashboard with stats and charts  
🎨 Clean and responsive UI  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript, Chart.js  
- **Backend:** Python (Flask)  
- **Libraries:** Flask-CORS, JWT, Requests
